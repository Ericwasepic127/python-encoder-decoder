# python-encoder-decoder
Python Encoder and Decoder function
# Free encoding and decoding function 
Use `simple`
> Just works through!
# Use as:
```python
from coder import encoder, decoder

text = "This is sample text"
# encoding text

key = 19373693
# any key you want

encoded = encoder(text, key)
print("Encoded!", encoded)
# displays encoded

decoded = decoder(encoded, key)
print("Decoded!", decoded) # displays decoded

print("Matches original:", decoded == text)
# will be True
```
